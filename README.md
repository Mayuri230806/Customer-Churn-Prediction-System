# Customer-Churn-Prediction-System
A Machine Learning based web application that predicts whether a customer is likely to stay or leave a telecom company. The project is developed using Flask for the backend and a modern responsive UI for an interactive user experience.

## Features

- Customer churn prediction using Machine Learning
- Probability score for prediction
- Risk level classification
- Modern responsive user interface
- Real-time prediction with Flask
- Clean and user-friendly design

## Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML5
- CSS3
- Bootstrap

## Machine Learning Model

- Logistic Regression
- Data Preprocessing
- One-Hot Encoding
- Feature Engineering
- Model Serialization using Joblib
